Build a JavaScript Snake Game


After finish learning JavaScript in university, I think it's time to pick some simple projects to reinforce the knowledge and programming skills.


To successfully implementing a Snake Game, I have watched tutorials in YouTube and read some blogs of developing Snake Game in different way. I learned the best part of their code, and finally this is what I got. GitHub Code


For the learner who has basic knowledge of JavaScript, what you should do first is learning the HTML Canvas (through W3C). after 3 minutes you should have no trouble reading code about Canvas, then you can start to design the algorithm of Snake Game (maybe write down the pseudo code) without read blog's code, or think of the question, how to let a Snake Game successfully run by JavaScript?


First, initialise/draw a play field for Snake Game in the webpage run by Canvas, basically 2 methods, can be implemented by drawing a HTML table in the webpage that every table cell represents a grid in game, or use coordinate axis directly. The next step, we set up the random function of candy’s appearance in the play field, this is simple, the key point is not to occupy same grid/pixel as snake do and not exceed the edges of play field. Then we come towards the core part, or said the hard part, how to make the Game run. Based on the functions given by JavaScript original package, we can used setInterval (interval, function()) to refresh the play field every interval time and run the given function. If the interval is short enough, we can observe the snake dynamically moved in the playfield. Now let's begin. We assume the length of snake is N, one problem is, with the fixed frequency (interval), how to make snake act like move forward (fill the grid next to the snake and eliminate grid after its tail)? The solution is, treat snake as array, and in every interval, we update its array elements (update its coordinate), we can used shift(), unshift() methods to do it. If we “eat” the candy, we should also update snake’s length for next interval. That’s roughly how the Snake Game run.
