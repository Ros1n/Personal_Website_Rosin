Build a JavaScript Snake Game


After a period of JavaScript study, I think its time to pick some simple projects to practice the knowledge I learned.


To successfully implementing a Snake Game, I have learned tutorials in YouTube and went through some blogs telling of how they developed Snake Game in totally different ways. I learned the advantage of their code, and modified my own code, and finally this is how I got. Click here.


For the learner who has solid knowledge of JavaScript, what he should do first is learning the HTML Canvas (through W3C). after 3 minutes and he get understanding of Canvas, he should start to design or think of the process of Game without read others code, or said, how to design a Snake Game run by JavaScript?


First, we should initialise/draw a play field in the webpage with Canvas, there is basically 2 methods, can be implemented by either draw a HTML table in the webpage or use coordinate directly. Then we can set up the random function of candy’s appearance in the play field, this is simple, the key point is not to occupy same grid/pixel as snake do and not exceed the edges of play field. Then we come toward the core part, or said the hard part, how to make the Game run. Based on the functions given by JavaScript original package, we can used setInterval (interval, function()) to refresh the play field every interval time. If the interval is short enough, we can observed the snake dynamically moved in the playfield. We assume the length of snake is N, the problem is, with the fixed frequency (interval), how to make snake act like move forward (fill the grid next to the snake and eliminate grid after its tail)? The solution is, we can treat snake as array, and in every interval, we update its array elements (based on coordinate), we can used shift(), unshift() methods. If we “eat” the candy, we should also update snake’s length for next interval. That’s roughly how the Snake Game run.
