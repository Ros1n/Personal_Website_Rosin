As a Learner How Can I get Favouriate/Free Developer Tools


As a beginner or learner of the programming, one question you must have considered, what developer tools should I used/can I used? As a recent Bechelor of Computer Science graduate who have used variety of developer tools, here I’d like to give some suggestions to the people who are confused with this question.


If you learned C/C++ in university, you must get familiar with Visual Studio. It is the top 1 recommendation, also the most popular IDE in the world. It contains both compiler and interpreter, its debugger can dynamically show the value of each variable in every step, which is very powerful, although it doesn’t support hot deployment as IntelliJ IDEA did. It is great for GitHub user, because the user can directly clone repositories from remote to the local through VS, and update code in the remote through it, way easier than the git bash. To the people who used ASP.NET platform technologies, it’s also the best choice. Microsoft provide variety of developer tools and services in VS, you can develop mobile device app, desktop app and web form app in Visual Studio. Even for some languages or technologies that VS is not designed for it, for example, Python, after installing some plugins, you can still develop and debug it in VS. the Visual Studio cost about $500 a year, but the good news is, the Microsoft DreamSpark (should called it Microsoft Imagine now) give student free, full-featured Visual Studio! You just need to give your university email, then you will get a free code. Thanks to Microsoft!


If you career path is to be a Java Developer, then you must know the Eclipse, which is free, fast and awesome IDE. You can install some plugins to make it suits more technologies, for example, Android, so you don’t need to install Android Studio. It supposed to be enough to handle all Java tasks in campus. In the work you may require using the professional developer tools like MyEclipse or IntelliJ IDEA, but don’t worry, the company will buy it for you.


If you are currently study or have plan to study Python because recently Machine Learning is so hot, then you must not miss this guide for picking the suitable developer tools. If you develop small Python applications, not have several separate files or packages, I recommend Sublime Text3, is free and easy to use, you can add plugins to make it suits more languages like JavaScript or PHP. For the people develop large Python projects that require some frameworks, I recommend used IntelliJ IDEA for Python. Worried about high cost fee? No worries, it has free version for student.


If you are confident in HTML or CSS, you may decide to write your own website or blog. One necessary step is to buy a domain name, it normally costs $5 to $20 per year. However, if you don’t mind your domain name contain some company name, like “http://your_name.github.io” or “http://your_name.azurewebsites.net”, you can deploy your website on GitHub or Azure for free.


Ok, I guess that’s some of the experience I like to share, if you have questions, you can email me at: xluo@protonmail.com , hope you find your favourite developer tools.
