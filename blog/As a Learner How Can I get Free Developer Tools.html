As a Learner How Can I get Free Developer Tools


As a beginner or learner of the programming, one question you must considered, what developer tools should I used/can I used? As a recent Bachelor of Computer Science graduate who have used variety of developer tools, I’d like to give suggestions to the people who are confused with this question.


If you ever learned C/C++ in university, you must be familiar with Visual Studio. It is the most popular IDE in the world, also my top 1 recommendation. It contains both compiler and interpreter and its debugger can dynamically show the value of each variable in every step, which is very powerful, although it doesn’t support hot deployment as IntelliJ IDEA did. It is great for GitHub user, because the user can directly clone repositories from remote to the local through VS, and update code to the remote through it, way easier than the git bash. For people who used ASP.NET platform technologies, it’s also the top choice. Microsoft provide variety of developer tools and services in VS, you can develop either mobile device app or desktop, even web form app in Visual Studio. Even for some languages or technologies that is not originally supported by VS, for example, Python, you can still develop and debug it in VS after installing Python Plugins. The Visual Studio normally cost about $500 a year, but the good news is, the Microsoft DreamSpark (should called it Microsoft Imagine now) give student free, full-featured Visual Studio! You just need to give your university email, then you will get a free code. Thanks to Microsoft!


If you plan to be a Java Developer, then you must know the Eclipse, which is free, fast and easy to use. You can also use Eclipse to develop other languages by installing plugins, for example, Android, so you don’t need to install Android Studio. The original Eclipse version is supposed to be enough for all Java tasks in campus. In work you may be asked to use the enterprise version develop tools, like MyEclipse or IntelliJ IDEA, but don’t worry, the company will buy it for you.


If you are currently study or have plan to study Python because recently Machine Learning is so hot, then you must not miss this guide for picking suitable IDEs for Python. If you develop small Python applications, with only several code files and no complex directory structure, I recommend Sublime Text3, is free and easy to use, you can install plugins to make it suits more languages like JavaScript or PHP. For the people develop large Python projects that require some frameworks and more packages independencies, I recommend IntelliJ IDEA for Python. Worried about high cost fee? No worries, it has free version for student.


If you are confident in HTML or CSS, you may decide to write your own website or blog. The easiest way is to create website/blog inside develop tools like WordPress or Hexo. But some people may want to go through the whole process. For both two, one necessary step is to buy a domain name, it normally costs $5 to $20 per year. However, if you don’t mind your domain name contain some company name or technique name, like “http://your_name.github.io” or “http://your_name.azurewebsites.net”, you can deploy your website on GitHub or Azure for free.


Ok, I guess that’s all, if you have any questions, you can email me at: xluo@protonmail.com. Hope you find your favourite developer tools.
